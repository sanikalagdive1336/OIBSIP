# OIBSIP - Data Science Internship

This repository contains the tasks completed as part of the **Oasis Infobyte Data Science Internship**. The projects demonstrate the application of Python, data analysis, data visualization, and machine learning techniques to solve real-world problems.

---

## 📝 Task 1: Iris Flower Classification

### Objective
Build a Machine Learning model to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

### Workflow
- Data Loading
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Train-Test Split
- Model Training
- Model Evaluation

### Models Used
- Logistic Regression
- Decision Tree Classifier

### Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1-Score
