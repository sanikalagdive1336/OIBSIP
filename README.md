# OIBSIP - Data Science Internship

This repository contains the tasks completed as part of the Oasis Infobyte Data Science Internship.

## Task 1: Iris Flower Classification

### Objective
Build a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica.

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

### Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Visualization
- Train-Test Split
- Model Training
- Model Evaluation

### Models Used
- Logistic Regression
- Decision Tree Classifier

### Evaluation Metrics
- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-Score
